# Your Deployment Pipeline Ends at a Firewall Ticket

**Conference:** Grassr00tz 2026  
**Speaker:** Tyler Coady  
**Date:** 2026

## Overview

This talk explores the gap between automated deployment pipelines and real-world infrastructure constraints, specifically how network security policies and firewall rules often become the final bottleneck in modern CI/CD workflows.

## Key Topics

- **The Pipeline-to-Production Gap**: How mature deployment automation still hits infrastructure walls
- **Firewall Policies as a Blocker**: Understanding firewall tickets as a critical path item in deployments
- **Security vs. Automation**: Balancing DevOps velocity with network security requirements
- **Solutions & Best Practices**: Strategies to reduce deployment delays caused by infrastructure tickets
